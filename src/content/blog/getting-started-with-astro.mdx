# Getting Started with Astro

Astro is a modern static site framework that lets you build fast, content-driven websites.

## Why Astro?

- **Zero JS by default** — Astro ships zero JavaScript by default
- **Framework agnostic** — Use React, Vue, Svelte, or any combination
- **Content collections** — Type-safe content management with MDX support

## Basic Structure

```astro
---
import Layout from '../layouts/BaseLayout.astro';
---

<Layout title="Hello World">
  <h1>Hello, World!</h1>
</Layout>
```

Astro makes it easy to build performant websites without sacrificing developer experience.
