# 你好，世界！

歡迎來到 DinoYee 的部落格！這是使用 **Astro** + **React** + **MDX** 搭建的第一篇文章。

## 關於這個部落格

這個部落格使用以下技術：

- [Astro](https://astro.build/) — 靜態網站框架
- [React](https://react.dev/) — UI 元件庫
- [MDX](https://mdxjs.com/) — Markdown + JSX
- [Tailwind CSS](https://tailwindcss.com/) — 樣式框架
